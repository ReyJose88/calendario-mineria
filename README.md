# Calendario minero

## Requisitos

- Node.js 14+ 
- npm o yarn

## Instalación

1. **Clonar el repositorio**
```bash
git clone https://github.com/ReyJose88/calendario-mineria.git
cd calendario-mineria
git checkout main
```

2. **Instalar dependencias**
```bash
npm install
```

## Uso

```bash
npm run dev
```
La aplicación se abrirá en `http://localhost:5173`
